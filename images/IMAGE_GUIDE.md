## üìù Image Optimization Tips

1. **Resize Before Upload**
   - Use specified dimensions above
   - Larger = slower website

2. **Compress Images**
   - Use TinyPNG or Squoosh
   - Aim for < 200KB per image

3. **Use Correct Format**
   - JPG for photos
   - PNG for logos/graphics with transparency
   - WebP for modern browsers (optional)

4. **Name Files Properly**
   - Use descriptive names
   - Include keywords: "delhi-packers-movers-truck.jpg"
